# 智慧树自动互动
智慧树自动刷互动分
## 如何使用
1.登录智慧树<br/>
2.打开一次问答和成绩分析<br/>
3.进入我的学堂按下f12<br/>
4.在网络里面找到getStudentStudyInfo（其他的get也行），复制里面的cookie<br/>
5.打开cookie.txt将cookie复制进去<br/>
6.其他参数根据自身情况修改（question_number是提问个数，answer_nunber是回答个数）
7.运行py<br/>

## 注意事项<br/>
1.请事先pip requests,json<br/>
2.出bug提交issues，不定时会看<br/>
